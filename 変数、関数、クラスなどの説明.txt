<DataBase>

class		DataBase 　　様々な初期設定をするクラス
String[]	objects          敵のクラス名が入る。ReadTextの、敵クラス名が入った配列を複製する
float		scwhrate      	width/1600　これをvxなどにかけると画面の大きさに比例した速度になる
int			bs                 通常の弾の弾速
int			screenw         画面の横の長さ
int			screenh			画面の縦の長さ　screenwと、横の長さとの比から求める

String	erase			消すときの効果音のファイル名

HashMap<String, MyObj>	
			oriEnemys     	敵を作るときに模範となるオブジェクト
Player	oriplayer        プレイヤーの模範となるオブジェクト

setobjectnames()		敵クラス名（ReadTextの配列の複製）をobjectsに入れ、oriEnemysの要素をその配列の大きさ分生成する

setsound(String object, String command, String filename)
								敵の効果音をoriEnemysに入れる

setobjects()				敵・プレイヤーの初期設定を模範オブジェクトに設定する

setImage(MyObj e, String filename)
								渡されたオブジェクトの画像用変数の最初の要素に、
								渡されたファイル名の画像を入れる
								
Reverse(Image img)	渡された画像を反転する

reSize(PImage img, int w, int h)
								渡された画像を(w, h)の大きさにする（拡大・縮小）

<MyObj>
class MyObj	敵・プレイヤーオブジェクトの基底クラス
float 	x, y		画像左上の座標
float	vx			横方向の速度
int		w, h		画像の大きさ
int		energy	粉エネルギー
int		hp			体力
int		Bcount	弾用時間カウント
int		count	汎用カウント
